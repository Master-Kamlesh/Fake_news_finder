What is Fake News Detector?

Fake News Detector is an AI-powered tool designed to help identify misinformation, sensationalized content, and unreliable news articles. Using a combination of rule-based heuristics and machine learning models, the system analyzes text for signs of fake news with high accuracy.

How It Works

1. Rule-Based Analysis

The system scans for common red flags associated with misinformation:

  • Sensational Language: Words like "shocking," "unbelievable," "you won't believe"
  • Punctuation Abuse: Excessive exclamation marks or question marks
  • CAPS LOCK Overuse: Abnormally high ratio of capital letters
  • Clickbait Patterns: Common fake news templates and structures
  • Text Anomalies: Very short or unusually long articles

2. Machine Learning Analysis

Optional transformer-based deep learning models analyze the semantic meaning of text to detect misinformation patterns that may not be obvious from surface-level features.

3. Scoring System

The system produces a fake-news score from 0 to 1:

  • 0.0 - 0.3: Likely Real News
  • 0.3 - 0.7: Uncertain / Mixed Signals
  • 0.7 - 1.0: Likely Fake News

Technology Stack

This tool is built with:

  • Backend: Python, Flask
  • Machine Learning: Transformers (Hugging Face), Scikit-Learn
  • Web Scraping: BeautifulSoup4, Requests
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Deployment: Vercel, Heroku, or Google Cloud Run (free tier)
